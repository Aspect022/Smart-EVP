# Code of Conduct
## Our commitment
We want SmartEVP+ collaboration to be respectful, inclusive, and constructive for everyone participating in issues, pull requests, reviews, and discussions.

## Expected behavior
- Be respectful and professional in all interactions.
- Assume good intent and focus feedback on code and outcomes.
- Share actionable technical feedback with context.
- Help newcomers by pointing to docs and reproducible steps.

## Unacceptable behavior
- Harassment, discrimination, or personal attacks.
- Trolling, insults, or inflammatory comments.
- Publishing private information without consent.
- Repeatedly derailing technical discussion.

## Enforcement
Repository maintainers may moderate discussions, remove harmful content, or restrict participation when behavior violates this code.

## Reporting
If you experience or witness unacceptable behavior, report it privately to repository maintainers through repository owner contact channels.
